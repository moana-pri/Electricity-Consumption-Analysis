India Electricity Consumption Analysis (2019–2020)

This project analyzes electricity consumption patterns across Indian states from 2019 to 2020, including the impact of the COVID-19 lockdown period (March–June 2020).

Using SQL and Tableau, the project visualizes monthly trends, regional demand differences, and post-lockdown recovery patterns through interactive dashboards and story points.

📌 Key Insights

Electricity demand dropped significantly during lockdown (March–June 2020).

Industrial states showed higher demand fluctuations.

Gradual recovery in electricity usage after lockdown restrictions eased.
