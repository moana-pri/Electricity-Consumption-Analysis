# Electricity Consumption Analysis in India

This project analyzes electricity consumption patterns across Indian states from **2019 to 2020**, including the impact of the **COVID-19 lockdown period (March–June 2020)**.

## Technologies Used

* SQL
* Tableau
* Flask
* Data Visualization

## Project Features

* Monthly electricity consumption trend
* Region-wise demand comparison
* Lockdown impact analysis
* Post-lockdown recovery analysis

## Dataset

The dataset contains state-wise electricity consumption across India with geographic coordinates and dates.
